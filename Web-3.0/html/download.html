GLoBES Download

Software Requirements

GLoBES is currently available for Linux and Mac OS under the GNU Public License. Building the GLoBES binaries requires that the following tools and libraries are installed:
  • make and gcc
  • GNU Scientific Library
These tools are included in many Linux distributions. Otherwise they can easily be downloaded from the provided links.

GLoBES Software Download

GLoBES can be downloaded as a single gzip-ed tar ball. The latest up-to-date version is here:
  • globes-3.0.tar.gz  (Jan. 23, 2007)

What's new in version 3.0?

  • Customizable treatment of systematical error
  • User-defined priors to incorporate arbitrary external information on the oscillation parameters
  • An interface for the implementation of non-standard physics
  • Beta-beam fluxes are now built-in
  • New and updated AEDL files
  • Various bug fixes
  • Improved performance
GLoBES documentation and related material can be found under the menu item Documentation. The convention for the version number is as follows:
- the first digit indicates major software revisions
- the second digit is for changes when new features are introduced
- the last digit is for minor bug-fix releases. An up-to-date list of known bugs and bug fixes is here

For compatibility we provide als some previous versions of GLoBES:

  • globes-2.0.11.tar.gz  (Mar. 14, 2005)
  • globes-2.0.1.tar.gz  (Aug. 10, 2004)
  • globes-2.0.0.tar.gz  (Jul. 29, 2004)

GLoBES - Pre-Defined Experiment Files

The most important ingredient of GLoBES are adequate experiment descriptions. The experiments posted here may serve as examples in using AEDL, but they may also serve as starting point for modifications. All pre-defined AEDL files contain in their header the references to their origin. The GLoBES Team would be happy to include your glb-files into this list. Please mail to globes@ph.tum.de
