GLoBES - Pre-defined AEDL-Files

sys-T2HK

The sys-T2HK superbeam experiment is an upgrade of the T2K experiment. It can be simulated with the file sys-T2HK.glb. Please check the header of the files for references of their origin. Please do not forget to cite these references if the file sys-T2HK.glb is used for a scientific publication or a talk. Check the appendix of the GLoBES manual for more details of the experiment description.

Author(s)

Patrick Huber, Mauro Mezzetto, Thomas Schwetz
