DoubleCHOOZ

The DoubleCHOOZ reactor experiment can be simulated with the files D-Chooz_near.glb and D-Chooz_far.glb. Please check the header of the files for references of their origin. Please do not forget to cite these references if the files D-Chooz_near.glb and D-Chooz_far.glb are used for a scientific publication or a talk. Check the appendix of the GLoBES manual for more details of the experiment description.
