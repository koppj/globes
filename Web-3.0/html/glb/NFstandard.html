GLoBES - Pre-defined AEDL-Files

Standard Neutrino Factory

A standard neutrino factory experiment can be simulated with the file NFstandard.glb. Please check the header of the files for references of their origin. Please do not forget to cite these references if the file NFstandard.glb is used for a scientific publication or a talk. Check the appendix of the GLoBES manual for more details of the experiment description.
