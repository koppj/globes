GLoBES - Pre-defined AEDL-Files

SPL

The SPL superbeam experiment can be simulated with the file SPL.glb. Please check the header of the files for references of their origin. Please do not forget to cite these references if the file SPL.glb is used for a scientific publication or a talk. Check the appendix of the GLoBES manual for more details of the experiment description.

Author(s)

Thomas Schwetz-Mangold, used for Campagne et al., hep-ph/0603172
