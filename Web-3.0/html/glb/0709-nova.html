GLoBES - Pre-defined AEDL-Files

NOνA as of July 2009

The NOνA superbeam experiment can be simulated with the file 0709-nova.glb. Please check the header of the files for references of their origin. Please do not forget to cite these references if the file 0709-nova.glb is used for a scientific publication or a talk. Check the appendix of the GLoBES manual for more details of the experiment description.

Author(s)

P. Huber, M. Lindner, T. Schwetz and W. Winter
