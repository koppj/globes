GLoBES - Pre-defined AEDL-Files

Medium Gamma β-Beam

A medium gamma green-field β-beam scenario can be simulated with the file BB_350.glb. Please check the header of the files for references of their origin. Please do not forget to cite these references if the file BB_350.glb is used for a scientific publication or a talk. Check the appendix of the GLoBES manual for more details of the experiment description.

Author(s)

The GLoBES Team, following Setup III from Burguet-Castell et al., Nucl.Phys.B725:306-326,2005
