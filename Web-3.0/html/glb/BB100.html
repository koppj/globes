GLoBES - Pre-defined AEDL-Files

Low Gamma β-Beam

The CERN to Fréjus β-beam baseline scenario can be simulated with the file BB_100.glb. Please check the header of the files for references of their origin. Please do not forget to cite these references if the file BB_100.glb is used for a scientific publication or a talk. Check the appendix of the GLoBES manual for more details of the experiment description.

Author(s)

Thomas Schwetz-Mangold, used for Campagne et al., hep-ph/0603172
