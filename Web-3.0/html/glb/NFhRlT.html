GLoBES - Pre-defined AEDL-Files

Variable Neutrino Factory - High Resolution - Low Threshold

A variable neutrino factory experiment with a hybrid detector at a higher energy resolution and a lower energy threshold in the appearance channels can be simulated with the file NF_hr_lT.glb. Similar to the variable neutrino factory in NFvar.glb the disappearance channels in NF_hR_lT.glb are defined without CID and the MINOS threshold is used. The parent energy of the muons and the baseline can be set up with the AEDL variables emax and BASELINE. Please check the header of the files for references of their origin. Please do not forget to cite these references if the file NF_hR_lT.glb is used for a scientific publication or a talk. Check the appendix of the GLoBES manual for more details of the experiment description.
